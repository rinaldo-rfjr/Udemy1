Git com Cmder/Github

git config --global credential.helper 'store' -> Acesso com git-credentials. Após o primeiro push ou pull, o git armazenará usuário/senha
git config --global credential.helper 'store --file filename' -> Acesso com git-credentials. Credenciais armazenadas no arquivo indicado.
git config --global --unset credential.helper -> Remove a configuração de acesso com git-credentials
git config --global --unset credential.helper -> Remove a configuração de acesso com git-credentials
git init -> Prepara a pasta para ser um repositório local
git status -> Situação do repositório
git add {filename} -> Adiciona o arquivo ao controle de versão
git add . -> Adiciona todos os arquivos modificados ao controle de versão
git commit -> Salva as alterações no repositório local
git commit -m 'texto do commit' -> Salva as alterações no repositório local já com um texto de descrição
git commit -f -> Força a exclusão no repositório local
git rm -> Exclui as alterações no repositório local
git revert -- {filename} -> Desfaz as alterações do arquivo
git push --set-upstream origin master -> Envia as alterações ao repositório remoto. Nesse exemplo, master.
git pull origin {branch_name} -> Sincroniza o código local com o código remoto na branch indicada.
git pull origin master --allow-unrelated-histories -> Em caso de conflitos no push, sincroniza o repositório local com o remoto.
git log -> Exibe o histórico de operações na Branch. O mesmo que examinar os commits no Github.
git branch -> Lista todos os branchs catalogados. A branch em verde é a branch ativa.
git branch -a -> Lista todos os branchs catalogados, locais e remotos.
git branch -D {branch_name} -> Remove a branch local.
git checkout {branch_name} -> Ativa a branch informada.
git checkout -b {branch_name} -> Cria e ativa uma nova branch local.
git push -u origin {branch_name} -> Cria a branch no repositório remoto e sincroniza o trabalho.
git config --get core.excludesfile -> Informa onde está o arquivo .gitignore
git config --global core.excludesfile '~/.gitignore' -> Configura o arquivo .gitignore
